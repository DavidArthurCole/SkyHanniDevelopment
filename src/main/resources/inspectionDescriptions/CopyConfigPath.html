Highlights properties annotated with @ConfigOption and offers a quick fix to copy their full config path to the clipboard.

Resolves the dotted path (e.g. inventory.items.slot) by walking up the config class hierarchy, stripping the modules.editor. prefix where applicable. 


@ConfigOption(name = "My Option", desc = "")
var myOption: Boolean = false

Trigger the quick fix via Alt+Enter on the property name to copy the path.
